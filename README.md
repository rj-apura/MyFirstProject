# Lab: Reading Products from a File

This program reads data from a text file ad offers the ability to search for products.

## How to Run
1. Open the Program in your IDE
2. Create a .txt file containing the products you offer in separate lines
3. View product list
4. Optional: Perform a product search to verify availibility

## Example Output 
Product List:
Apple
Banana
Orange
Milk
Bread
Eggs
Rice
Cheese
Juice
Butter
Total products: 10
